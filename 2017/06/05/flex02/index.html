1.下面，就来看看Flex如何实现，从1点到9点骰子的布局

参考的主要资料https://davidwalsh.name/flexbox-dice

1.1一个点的骰子布局。

1.flex布局flex－direction默认主轴方向是水平方向，起点在左端
2.flex布局justify-content默认项目在主轴的对齐方式左对齐
3.结合使用justify-content改变项目主轴的对齐方式；结合使用align－item改变项目交叉轴上的对齐方式

 .box{

    display: flex;
    justify-content:center;
    align-items: center;

}

1.2两个点的情况


实现思路

<div class="box">
    <span class="item"></span>
    <span class="item"></span>
</div>

.box{

    display: flex;
    justify-content:space-between; 
}
.box .item:nth-child(2){
    align-self: flex-end;
}

1.3三个点的情况


实现思路

在二点的基础上添加一个点设置该点align－self为flex－end

.box{

    display: flex;
    justify-content:space-between; 
}
.box .item:nth-child(2){
    align-self: center;
}
.box .item:nth-child(3){
    align-self: flex-end;
}

2.接着介绍一下flex布局在实际开发中的应用

参考的主要资料:http://www.ruanyifeng.com/blog/2015/07/flex-examples.html

2.1更棒，更简洁的栅格系统

现今大部分栅格系统都使用了两种布局方式中的一种：浮动 (float) 或者 內联块 (inline-block)


 <div class="source">
    <div class="Grid mt_20">
        <div class="Grid-cell">
            <div class="grid-content bg-purple-dark"></div>
        </div>
    </div>
</div>

*{
    box-sizing: border-box;
}
.source{
    width: 809px;
    padding: 24px;
    border: 1px solid #eaeefb;
    border-radius: 4px;
    margin: 0 auto;
}
.bg-purple-dark{
    background: #99a9bf;
}
.bg-purple{
    background:#d3dce6;
}
.grid-content{
    min-height: 36px;
    border-radius: 5px;
}
.mt_20{
    margin-bottom: 20px;
}

2.1.1基础栅格系统

没有制定特定的宽度，它们自然的平分每一行的空间并撑满每一行，并且高度默认为0。

.Grid{
    display: flex;
}
.Grid-cell {
  flex: 1;
}
间隔布局

.Grid-gutters {
  margin: -1em 0 1em -1em;
}
.Grid-gutters > .Grid-cell {
  padding: 1em 0 0 1em;
}

2.1.2独立的尺寸

当你的需求不是等宽栅格的时候，可以添加尺寸属性到特定的栅格中。没有尺寸属性的栅格将简单地继续平分剩下的可用空间

.Grid-cell.u-full {
  flex: 0 0 100%;
}

.Grid-cell.u-1of2 {
  flex: 0 0 50%;
}

.Grid-cell.u-1of3 {
  flex: 0 0 33.3333%;
}

.Grid-cell.u-1of4 {
  flex: 0 0 25%;
}

2.3输入框布局

我们常常需要在输入框的前方添加提示，后方添加按钮。

<!-- appending -->
<div class="InputAddOn">
    <input class="InputAddOn-field">
    <button class="InputAddOn-item">…</button>
</div>

<!-- prepending -->
<div class="InputAddOn">
    <span class="InputAddOn-item">…</span>
    <input class="InputAddOn-field">
</div>

<!-- both -->
<div class="InputAddOn">
    <span class="InputAddOn-item">…</span>
    <input class="InputAddOn-field">
    <button class="InputAddOn-item">…</button>
</div>

.InputAddOn {
  display: flex;
  margin-bottom: 1.5em;
}

.InputAddOn-field {
  flex: 1;
}
.InputAddOn-field:not(:first-child) {
  border-left: 0;
}
.InputAddOn-field:not(:last-child) {
  border-right: 0;
}

.InputAddOn-item {
  background-color: rgba(147, 128, 108, 0.1);
  color: #666666;
  font: inherit;
  font-weight: normal;
}

.InputAddOn-field,
.InputAddOn-item {
  border: 1px solid rgba(147, 128, 108, 0.25);
  padding: 0.5em 0.75em;
}
.InputAddOn-field:first-child,
.InputAddOn-item:first-child {
  border-radius: 2px 0 0 2px;
}
.InputAddOn-field:last-child,
.InputAddOn-item:last-child {
  border-radius: 0 2px 2px 0;
}
